
# Purnatraders - Zerodha Auto Token Generator

1. Edit server.py and add your API_KEY and API_SECRET.
2. Push this folder to a GitHub repo named purnatraders.
3. Login to https://render.com
4. Create a New Web Service.
5. Connect your GitHub repo.
6. Deploy.

Redirect URL for Zerodha:
https://purnatraders.onrender.com/token
